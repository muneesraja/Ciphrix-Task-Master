Task Management Application (MERN Stack) 
Objective Build a full-stack Task Management Web Application using focusing mainly on 
the frontend (React.js). The goal is to create a clean, attractive, and responsive UI. 
Frontend (Main Focus) 
• Set up a React.js application using Material UI (MUI) or similar styling library 
(Shadcn). 
• Implement a Light/Dark theme switcher. 
• Design a modern, user-friendly, and responsive interface. 
• Create the following pages: Sign Up, Sign In, Dashboard (task list with 
pagination), and Add/Edit Task form. 
• Each task should include: Title, Description, Status (Pending/Completed), and 
Created Date. 
• Allow users to add and edit tasks. 
• Only Admin users should be able to delete tasks (hide delete option for normal 
users). 
• Connect frontend to backend and DB of your choice e.g. Node.js/Express + 
MongoDB backend using Axios. 
Backend 
• Build APIs for Sign Up, Sign In, Create, Update, Delete, and List Tasks. 
• Implement two user roles: Admin and Normal User. 
Submission 
• Submit your GitHub repository link containing both frontend and backend 
folders. 
• Share a live demo link (Vercel / Netlify)
